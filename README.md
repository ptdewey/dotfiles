# dotfiles
My essential configuration files and scripts  

## Installation Instructions
1. clone directory to home folder  
2. ```cd ~/dotfiles```  
3. run ```scripts/setup.sh``` which will create backups and symlinks  

## Dependencies
- anaconda
- zsh
- neovim
